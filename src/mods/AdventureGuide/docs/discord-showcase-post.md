Ever open the quest log and wonder where you're supposed to go? **Adventure Guide puts 170+ quests at your fingertips** with step-by-step walkthroughs, GPS navigation, and floating world markers — all without leaving the game.

- **Click any quest step or item source** and a GPS arrow guides you there — enemy drops, vendors, mining nodes, fishing spots, crafting recipes, all one click away
- **Floating markers above NPCs** show quest givers, turn-in targets, and enemies to hunt — with live respawn countdowns and night-spawn timers
- **Level estimates** help you tackle things in the right order, and your progress, item counts, and active step persist between sessions

**Quick Start**
Requires [BepInEx 5](https://github.com/BepInEx/BepInEx).

1. Install BepInEx if you haven't already
2. Download the mod from the [Adventure Guide page](https://erenshor-maps.wowmuch1.workers.dev/adventure-guide)
3. Place `AdventureGuide.dll` in `BepInEx/plugins/`
4. Press **L** in-game to open the full guide
5. Press **K** for a minimal quest tracker overlay

**Configuration**
All features are toggleable in [Configuration Manager](https://github.com/BepInEx/BepInEx.ConfigurationManager) (F1) or via `BepInEx/config/wow-much.adventure-guide.cfg`:

- **ShowArrow** — GPS arrow to navigation target (on)
- **ShowGroundPath** — ground path line via NavMesh (off)
- **ShowWorldMarkers** — floating quest icons above NPCs (on)
- **TrackerEnabled** — quest tracker overlay (on)
- **ReplaceQuestLog** — J opens Adventure Guide instead of quest log (off)
- **UiScale** — auto-detects from resolution, or set manually
