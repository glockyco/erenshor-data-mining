# Adventure Guide

BepInEx mod that adds an in-game quest guide overlay to Erenshor. Shows
step-by-step objectives, item tracking with live inventory counts, quest
chain navigation, and NPC/zone information.

## Architecture

```
src/
├── Plugin.cs              # Entry point, wires components, Harmony patches
├── PluginInfo.cs          # GUID, name, version
├── Config/
│   └── GuideConfig.cs     # Toggle key, quest log replacement
├── Data/
│   ├── GuideData.cs       # Loads quest-guide.json, provides lookup by DBName
│   └── QuestEntry.cs      # Deserialization classes for quest guide JSON
├── State/
│   └── QuestStateTracker.cs  # Mirrors GameData quest state via Harmony callbacks
├── UI/
│   └── GuideWindow.cs     # IMGUI split-panel window
└── Patches/
    ├── QuestAssignPatch.cs    # Postfix on GameData.AssignQuest
    ├── QuestFinishPatch.cs    # Postfix on GameData.FinishQuest
    ├── InventoryPatch.cs      # Dirty flag on inventory changes
    └── PointerOverUIPatch.cs  # Blocks game input over IMGUI window
resources/
└── quest-guide.json       # Embedded resource (generated by erenshor guide generate)
```

## How It Works

1. **Data**: `GuideData` loads `quest-guide.json` from embedded assembly resource at startup. Provides O(1) lookup by DBName and StableKey.

2. **State**: `QuestStateTracker` syncs from `GameData.HasQuest` and `GameData.CompletedQuests`. Harmony postfixes on `AssignQuest`/`FinishQuest` keep it current. Dirty flag pattern for lazy UI updates.

3. **Step Detection**: Current step derived from live game state -- item counts from `GameData.PlayerInv.StoredSlots`, zone from scene name. Steps that can't be detected (talk, kill) default to showing as active.

4. **Input Isolation**: `PointerOverUIPatch` patches `EventSystem.IsPointerOverGameObject()` to return true when cursor is over the IMGUI window. Tracks drag state to handle fast drags where the cursor outruns the window. Skips hit-testing when `Cursor.lockState` is locked (game locks cursor to screen center during camera drag).

## Data Pipeline

```bash
uv run erenshor guide generate    # Reads clean DB, produces quest_guides/quest-guide.json
```

The generator auto-creates steps for ~90% of quests (fetch, kill, dialog, zone, shout, item read). The remaining ~10% (scripted events, complex chains) need manual curation via `quest_guides/manual/*.json`.

After regenerating, update the embedded resource:

```bash
cp -f quest_guides/quest-guide.json src/mods/AdventureGuide/resources/
```

## Configuration

`BepInEx/config/wow-much.adventure-guide.cfg`:

| Setting | Default | Description |
|---------|---------|-------------|
| ToggleKey | L | Key to open/close the guide window |
| ReplaceQuestLog | false | J key opens Adventure Guide instead of game quest log |

See the `mod-development` skill for shared build/deploy/hot-reload workflow.
